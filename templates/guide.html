📖 사용법 안내

스마트오더 라벨 도구를 처음 쓰는 분도 바로 따라 할 수 있게 정리했습니다.

리테일러(CU·GS·이마트24)에서 받은 스마트오더 RAW 엑셀을 올리면, 우리(한익스) 기준으로 센터명 → 이고센터, 상품코드 → 한익스상품명을 매핑해서 부착양식(A4) 라벨과 TOTAL 표지를 만들어 줍니다.

① 채널 선택 · RAW 업로드
→
② 미매핑 등록(센터·상품)
→
③ 라벨·표지 출력

단계별 사용법

1

채널 선택하고 RAW 파일 올리기

라벨생성 메뉴에서 채널(CU / GS / 이마트24)을 고르고, 리테일러 포털에서 내려받은 원본 엑셀을 올립니다.

  • 지원 형식: .xlsx / .xls (GS는 GS25·GS슈퍼 둘 다 자동 인식)
  • 파일을 고르면 초록색 ✅ 로 선택 확인이 뜹니다. 그 뒤 라벨 생성 → 버튼을 누르세요.
  • 올린 파일은 자동으로 보관되어 나중에 업로드이력에서 다시 열 수 있습니다.
💡 컬럼 이름으로 자동 인식하므로 열 순서가 달라도 됩니다. 센터명·상품코드·수량만 있으면 됩니다.
2

미매핑 코드 등록하기

처음 보는 센터·상품이 있으면 결과 화면에 미매핑으로 표시됩니다. 그 자리에서 바로 등록하면 다음부터 자동 적용됩니다.

  • 센터: RAW의 센터명 → 우리 이고센터(거점)를 드롭다운에서 선택
  • 상품: 상품코드 → 한익스상품명(표지에 찍히는 이름) + 구분(대월/프리미엄) 지정
⚠ 한익스상품명이 없으면 출력이 막힙니다. 표지에 빈 이름이 찍히는 걸 막기 위해서예요. 미매핑 상품을 모두 등록하면 바로 다운로드·인쇄됩니다.
3

라벨·표지 출력하기

  • ⬇ 부착양식 다운로드: 기존 양식과 동일한 A4 라벨 엑셀 (구분에 따라 대월/프리미엄 직매장 제목 자동)
  • 🖨 라벨 미리보기·인쇄: 화면에서 바로 확인하고 인쇄 (라벨 1장 = A4 1페이지)
  • TOTAL 표지: 화면에서 바로 보고 🖼 이미지 복사(카톡·엑셀 붙여넣기) 또는 PNG 저장·인쇄
💡 라벨 상품명은 화주(리테일러) 상품명 그대로, 표지·집계는 한익스상품명으로 표시됩니다.

부가 기능

🗂

마스터 관리

센터·상품 매핑을 화면에서 한 건씩 고치거나, 엑셀로 내려받아 한 번에 수정 후 업로드할 수 있습니다.

  • 센터마스터: 전 채널 공통 (센터명 → 이고센터). 한 번 등록하면 CU·GS·이마트24 모두 적용
  • 상품마스터: 상품 1개 = 1줄. 같은 상품이라도 채널마다 코드가 다르니 여러 코드를 쉼표로 넣으세요
  • 엑셀 업로드 시 추가·수정·삭제 내역을 먼저 보여주고 확인 후 반영(자동 백업)
📥

업로드 이력

올린 RAW 파일이 채널별로 누적 보관됩니다. 언제든 다시 열기 · 부착양식 재다운로드 · 원본 내려받기 · 삭제가 가능합니다.

📊

작업 집계 (검수리스트)

채널·일자를 선택하면 그 안을 배송센터(이고센터) → 센터명 → 상품명 순으로 묶어 수량을 집계합니다.

  • 센터명별 요약(소계) + 총합계 (엑셀 검수리스트와 동일한 형태)
  • 수량은 CU는 발주단위수량, GS·이마트24는 주문수량 기준
  • 클립보드 복사 · 엑셀 다운로드

자주 묻는 질문

Q. 다운로드 버튼을 눌렀는데 출력이 안 돼요.
한익스상품명이 등록되지 않은 상품이 있으면 막힙니다. 결과 화면의 미매핑 상품을 모두 등록하세요.
Q. 파일이 올라갔는지 모르겠어요.
파일을 고르면 업로드 칸이 초록색 ✅ + 파일명으로 바뀝니다. 그 뒤 라벨 생성 →을 누르면 결과 화면으로 넘어갑니다.
Q. GS25와 GS슈퍼를 따로 올려야 하나요?
아니요. GS 채널 하나에서 둘 다 자동으로 인식합니다.
Q. 센터/상품 이름이 조금 바뀌면 다시 등록해야 하나요?
센터는 센터명, 상품은 상품코드로 매핑합니다. 코드가 같으면 이름이 바뀌어도 유지됩니다.
